# Contributing

This project welcomes public-safe examples, templates, and documentation improvements.

## Good Contributions

- Better AI-readable website templates.
- More business FAQ examples.
- Safer schema.org metadata examples.
- Website audit checks.
- Clearer documentation for non-technical business owners.

## Do Not Contribute

- Private customer information.
- Real contracts, margins, or internal pricing.
- API keys, tokens, cookies, or webhooks.
- Guaranteed ranking, indexing, or revenue claims.
- Content copied from a client without permission.

## Review Standard

Every contribution should answer:

1. Does it make the website easier for humans and AI systems to understand?
2. Is the claim verifiable?
3. Is the sample public-safe?
4. Does it avoid ranking or revenue guarantees?
